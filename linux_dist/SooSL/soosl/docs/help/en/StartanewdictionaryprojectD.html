Start a new dictionary project (D)

Manage dictionary projects (D) ››

When you start a dictionary project, you need to do two things first, before you start adding signs:

  • Decide what you want in the dictionary
  • Get some video files

When you do those two things, you are ready to set up your dictionary project in SooSL.

After the dictionary project is set up, you can start adding signs to your dictionary.
