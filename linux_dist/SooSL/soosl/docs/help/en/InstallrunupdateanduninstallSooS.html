Install, run, update, and uninstall SooSL on Windows (D)

Get started with SooSL ›› Get started with SooSL Desktop (D) ››

Install SooSL Desktop on Windows

To install SooSL Desktop on Microsoft Windows, first download the installer. To do that, use your internet browser and go to https://soosl.net/software.html. Follow the instructions there; they tell you how to download and install SooSL. 

While you are there, also download the Demo ASL dictionary project.

Run SooSL Desktop on Windows

To start SooSL, double-click its desktop icon. If you don't have the icon on your desktop, then look in the start menu or search for "SooSL". 

You will see the startup dialog. (Read that help topic now.)

Update SooSL Desktop on Windows

If you are connected to the internet when you start SooSL, it will tell you if a new version is available. Then you can tell it to download and install the new version for you automatically.

You can also check for program updates while SooSL is running. Click Tools in the top-right corner and then click Update SooSL in the menu.  

To update to a new version manually, you normally do not need to uninstall or remove an old version of SooSL first. (Exception: if you want to install it in a new location, you will have to uninstall the old version first.) Just make sure SooSL is not running first. Then install the new version the same way as you installed SooSL the first time. SooSL will remember any previous program settings and where your projects are located.

Uninstall SooSL Desktop on Windows

You can uninstall SooSL like you would uninstall any other program. If you uninstall SooSL, this will not remove any dictionary projects. You may have to delete them manually.
