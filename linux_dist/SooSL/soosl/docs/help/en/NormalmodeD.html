In Normal mode, you can view signs. To add, edit, and delete signs, you need to enter edit mode.

Change the size of the window and the main sections

You can adjust the SooSL window in several ways:

  • To maximize the window (make it fill the whole screen), click the maximize button in the title bar. To restore the window so it fills only part of the screen, click the restore button in the same place. (These buttons look different on different computers. Sometimes they are on the left end of the title bar.)
  • To change the size of the window (if it is not maximized), click and drag its borders. 
  • To adjust the size of each of the main sections in the window, click and drag the section borders. 
  • To put all the main sections back to their original sizes, click Reset display.

The status bar

At the bottom of the SooSL window, you will see the status bar. It tells you useful things about the dictionary project, and gives you a couple useful controls.

  • The dialects that are included in the gloss list.
  • Whether the dictionary is locked (read-only) or editable (read-write). (See explanation in help for import a dictionary project and export a dictionary project.)
  • The location of the dictionary project's main JSON file. The folder is also where you will find the other project files (videos, etc.).
  • What version of SooSL you are using.
  • Control to change the display language.
  • Control to reset the column widths and main sections of the window to their original sizes.
