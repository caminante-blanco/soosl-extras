The startup dialog (D)

When you start SooSL Desktop, you will see a startup message for a few seconds. 

(If SooSL gets stuck on this dialog for a long time, like several minutes, please send us an email about the problem. If possible, include a screen shot of the dialog, which shows where it got stuck. Include your email address so we can ask you more questions. It can also help us if you send us the startup log file.)

After SooSL is done starting, you will see the main startup dialog.

You can do two things with this dialog:

  • Choose a display language. 
  • Choose a dictionary.

Read the sections below.

Choose a display language 

The display language is the language used in the SooSL menu, tooltips, dialogs, messages, etc. Sometimes people call it the "interface language". It can be different from the written languages used for glosses, translations and notes in a dictionary.

In the startup dialog, click the display language, then choose the language you want. 

             

We currently have English, Español (Spanish), Français (French) and Română (Romanian) available as display languages. We hope to add more languages in the future. (If you want to help us translate SooSL to use a new display language, please contact us.)

In the future, we also hope to have help files (like this one) in other languages.

Choose a dictionary 

You can also choose a dictionary. 

  • If you haven't used SooSL before, it will say "No dictionary". Just click . Then read the help topic about importing a dictionary. 

  • If you have used SooSL before, it will show the last dictionary you used. If you want to use that dictionary again, click to start working with it. 
  • If you want to use a different dictionary, click the dictionary title. 
  • You will see the Open Dictionary dialog. (Read that help topic for how to choose a dictionary to open.)
  • After you've chosen a dictionary to open, click to start working with it.
