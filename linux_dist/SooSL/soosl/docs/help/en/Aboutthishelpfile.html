About SooSL™ ››

We wrote this help file using simple English. We want many people to understand it.

This help file describes both SooSL Web and SooSL Desktop. 

  • Topics that describe only SooSL Web have a "(W)" in their title. 
  • Topics that describe only SooSL Desktop have a "(D)" in their title.
  • Topics that describe both programs have nothing extra in their title, neither "(W)" nor "(D)".

To decide what to read about, see the list of contents on the left.

  • The earlier topics tell you how to use SooSL to read and study dictionaries.
  • The later topics tell you how to make your own dictionary. 
  • We suggest you start reading here.

On the left, there are also tabs with an index of important words and a general search function.

If you don't find what you are looking for, please let us know and we will try to help. Also, please tell us how to make the help file better.
