About SooSL™

Help file version: 0.9.4 en (2025-07-10 16:47)

Overview in ASL

SooSL™ is a computer program for sign language dictionaries. 

You can use SooSL to read, study, or even make a dictionary:

  • You can read and study a sign language dictionary that someone else has made.
  • You can make a dictionary for any sign language. You can share it with other people or publish it on the web.

Both deaf and hearing people can use SooSL. 

  • Deaf people can use SooSL to learn a written language, or to learn new signs.
  • Hearing people can use SooSL to learn a sign language.
  • Deaf people (and their hearing friends) can make a dictionary. This will help both deaf and hearing people.

SooSL helps you make good quality dictionaries.

  • It uses videos for signs, example sentences, and explanations about the signs.
  • You can show more than one meaning for a sign or more than one sign for a meaning.
  • You can show meanings in one or more written languages.
  • You can show how different people use different signs (dialects).
  • You can show the grammar category for a sign.
  • You can find a sign if you know the written word.
  • If you don't know the written word, you can find a sign by parameters: handshape, location, motion, etc.
  • You can make different types of dictionaries.

SooSL is easy to use.

  • It uses a lot of icons, not just words.
  • When SooSL use words, it uses simple language.
  • The parts in words are available in English, Spanish, and other languages.

SooSL is actually two programs that work together closely: SooSL Web and SooSL Desktop.

  • In SooSL Web, you can use a dictionary that someone else has made and published on the SooSL website. You can use it on any computer, tablet, or smartphone that has a web browser (for example: Chrome, Edge, Firefox, or Safari).
  • In SooSL Desktop, you can use someone else's dictionary or make a new one. You can use SooSL Desktop on Windows, Mac, and some Linux computers. (See Computer requirements.) Later, you can publish your dictionary on SooSL Web and upload new versions.

This help file describes both programs. Topics that have "(W)" in their title are about SooSL Web. Topics that have "(D)" in their title are about SooSL Desktop. All other topics are about both programs.
