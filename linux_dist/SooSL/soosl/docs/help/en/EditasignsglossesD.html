Edit glosses for a sign (D)

Edit mode: Edit signs in a dictionary project (D) ››

You can add or edit glosses for a sign.

You can enter edit mode to add or edit glosses in two ways:

  • If you are adding a new sign, SooSL will first convert the sign's video. Then it will change to edit mode. You can then add glosses.
  • If you are editing an existing sign, first find it. Then enter edit mode: click Edit sign in the toolbar. After you are in edit mode, if you see parameters instead of words, click Words in the toolbar. 

You will see a white box for each written language in the dictionary project. (You can decide what written languages to put in the dictionary.)

     

Add a gloss for each written language in its white box. (If the written language has more than one way to express the meaning, you should add more than one gloss.)

The red color on the words means that you have not yet saved your changes.

To save your changes, click Save in the toolbar. 

If you are done making changes, exit edit mode to go back to normal mode.

If you have made changes and haven't saved them yet, SooSL will remind you to save them.
