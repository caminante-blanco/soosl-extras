SooSL Desktop

We keep lists of changes in SooSL Desktop on our website. 

  • Information about the newest version is on the software download page.
  • Information about older versions is on the older software page.

You can also find information on the current version in SooSL Desktop itself. Click on Tools > About SooSL, and then click on "Release Notes".

Finally, you can find these lists in the README.txt file included with the SooSL installation. It is in the following default locations:

  • Windows: C:/Program Files/SooSL/docs/README.txt
  • Mac OS X: Applications/SooSL.app/Contents/Resources/docs/README.txt
  • Linux: inside changelog.Debian.gz: /usr/share/doc/soosl/

SooSL Web

You can find the change log for SooSL Web in the main menu under About SooSL.

We do not use version numbers for SooSL Web, only dates to distinguish versions.
