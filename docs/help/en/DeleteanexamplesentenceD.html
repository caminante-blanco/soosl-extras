Delete an example sentence (D)

Edit mode: Edit signs in a dictionary project (D) ››

You can delete an example sentence for a sense, both its video and its translations. 

Be careful. After you delete the sentence, you can't undo it. If you want it back, you will have to add it again.

To delete an example sentence, first enter edit mode: click Edit sign in the toolbar. After you are in edit mode, if you see parameters instead of words, click Words in the toolbar. 

Click the trash can next to the sentence translations.

You will now see this:

The background is red, and there are lines through the words.

If you change your mind, click the trash can again.

To save your changes, click Save in the toolbar. 

If you are done making changes, exit edit mode to go back to normal mode.

If you have made changes and haven't saved them yet, SooSL will remind you to save them.
