Install, run, update, and uninstall SooSL on Mac (D)

Install SooSL Desktop on Mac

To install SooSL Desktop on Mac OS X or macOS, first download the installer. To do that, use your internet browser and go to https://soosl.net/software.html. Follow the instructions there; they tell you how to download and install SooSL. 

While you are there, also download the Demo ASL dictionary project.

Run SooSL Desktop on Mac

To start SooSL, double-click its icon in launchpad. If the icon is not showing in launchpad, then use Finder to find SooSL in the Applications folder and double-click its icon. If you still can't open SooSL with a double-click, then right-click (or double-finger-tap) its icon and choose "Open".

You will see the startup dialog. (Read that help topic now.)

Update SooSL Desktop on Mac

If you are connected to the internet when you start SooSL, it will tell you if a new version is available. Then you can tell it to download and install the new version for you automatically.

You can also check for program updates while SooSL is running. Click Tools in the top-right corner and then click Update SooSL in the menu.

To update to a new version manually, you normally do not need to uninstall or remove an old version of SooSL first. Just make sure SooSL is not running, then install the new version the same way as you installed SooSL the first time. SooSL will remember any previous program settings and where your projects are located.

Uninstall SooSL Desktop on Mac

You can uninstall SooSL like you would uninstall any other program. If you uninstall SooSL, this will not remove any dictionary projects. You may have to delete them manually.
