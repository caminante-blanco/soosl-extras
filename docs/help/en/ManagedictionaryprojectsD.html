Manage dictionary projects (D)

To make a dictionary, you need to use SooSL Desktop. You make a dictionary in a dictionary project. Here are some things you can do with a dictionary project:

  • Open a dictionary project
  • Start a new dictionary project
  • Add signs and edit signs in a dictionary project
  • Edit information about the whole dictionary project: 
    • general information
    • dialects
    • written languages
    • grammar categories 
  • Work together with other people on a dictionary project
  • Share a dictionary project with other people
  • Keep spare copies (backups) of your dictionary project
  • Delete a dictionary project
