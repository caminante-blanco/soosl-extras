Signs in a dictionary

SooSL starts in Normal mode. In Normal mode, you can read and study a dictionary.

In Normal mode, you can:

  • play videos of a sign
  • see the different meanings (senses) of a sign 
  • see example sentences that show how a sign is used
  • see the grammar category and dialect(s) of each sense
  • see notes about the sign and extra explanatory videos and images (pictures)
  • see the parameters that you can use for searching
  • search for a sign by its gloss, its parameters, or by dialect

To see what SooSL looks like in Normal mode, and what you can do with each part of the screen, read the following help topics:

  • Normal mode for SooSL Web. SooSL Web always runs in Normal mode. 
  • Normal mode for SooSL Desktop. SooSL Desktop starts in Normal mode, but you can also switch to edit mode if you want to change something.
