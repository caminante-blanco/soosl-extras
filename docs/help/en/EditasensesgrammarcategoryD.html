Change a sense's grammar category (D)

Edit mode: Edit signs in a dictionary project (D) ››

Each sense in a sign has a grammar category, such as "noun" or "verb". A sign can have more than one sense. Each sense can have a different grammar category. This happens when a sign can be used in different ways with different meanings. Each sense shows a different meaning and a different way to use the sign.

You can change the grammar category for a sense.

To do so, first enter edit mode: click Edit sign in the toolbar. After you are in edit mode, if you see parameters instead of words, click Words in the toolbar. 

You will see the grammar category for each sense right after the sense number.

To choose a new grammar category, click the box for the grammar category. You will see a list of possible grammar categories in the dictionary project. 

Choose one. If the choice you want isn't in the list, you can edit the grammar categories and add it.

To save your changes, click Save in the toolbar. 

If you are done making changes, exit edit mode to go back to normal mode.

If you have made changes and haven't saved them yet, SooSL will remind you to save them.
